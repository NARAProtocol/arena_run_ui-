## Design Context

### Users
Base-native NARA holders, board claimants, sponsors, and spectators using the arena as a live game surface rather than a passive dashboard. They need to understand the purse, race state, and their next action in seconds, while feeling like they entered a public combat venue with real stakes.

### Brand Personality
Aggressive, ritualized, and credible. The interface should feel like a live race-control broadcast for a brutal on-chain contest, not a casino landing page and not a generic DeFi admin panel.

### Aesthetic Direction
Dark-only tactical arena broadcast. Industrial race telemetry, ember-and-gold hazard palette, oversized display typography, dense command controls, and a visible central track stage. Avoid neon cyberpunk, gradient-heavy AI aesthetics, pastel softness, or glossy game launcher tropes.

### Design Principles
1. The track is the center of gravity. The page must feel like a race first and a dashboard second.
2. Every data point should support tension: purse, cull timing, overdrive, active runners, and burn pressure.
3. Controls should read like a command deck with clear risk and force, not generic form buttons.
4. Locker economics must stay legible, but they are support context for the spectacle, not the visual focal point.
5. Motion should reinforce heat and urgency without making the interface noisy or unreadable.
